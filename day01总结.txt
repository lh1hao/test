1.JDK安装，配置JAVA_HOME,及检验。
2.程序编写
	程序员写的叫源代码。
	源代码不能运行
	需要先编译成.class文件才能运行 
	.java  ----  .class  通过   javac 文件名.java    的方式进行编译
	可以使用    java 文件名(不带扩展名) 的方式进行程序运行
3.注释
	解释说明程序的文字，不会被运行。 是每一个程序员都必须具有的良好习惯
	单行 //
	多行  /*  */
	文档  /** */
4.关键字
	被Java赋予特殊含义，有着专门用途的单词。 public class staitc void  true false 必须能读 能写
5.标识符
	自己起名字的地方
	组成元素
		字母 数组 _ $
	规则
		数字不能开头
		不能使用关键字
		严格区分大小写
		
	Java中不同部分的命名规则
	类名:多个单词连接时，每个单词的首字母都大写
	变量&方法名：多个单词连接时，第一个单词的首字母小写，其余单词首字母大写
6.常量
	不可改变的数值，每一个数值都有其相对应的数据类型
	整数  0 1 2
	浮点  0.1 1.0 2.2
	布尔  true false	只有两个值，满足条件true 不满足条件fasle
	字符  'a'		不能为空，只能有一个字符 
	字符串常量 "" 任意长度
